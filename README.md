# Indivisual
## Overview
個人開発とSwitch2対応から，Indivisual(個人)とInput(入力)，di(2)，Visual(可視化)を意味してIndivisualの名前になりました

コントローラ入力をPCに表示します．遅延0でゲーム機と接続しながら，PC側に表示することが可能です．Switch Proコントローラ，およびSwitch 2 Proコントローラに対応します．

## Usage
予めIndivisual用のFWを書き込み，USBスニッフィング可能なRP2040を用意します．対応するCOMポート選択し，利用するコントローラを指定します．Connectボタンで接続します．
